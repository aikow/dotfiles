# BASH

## Custom Prompt
Add
```
export PS1="\e[0;32m[\e[0m\e[0;31m\u\e[0m\e[0;33m@\h\e[0m \e[0;34m\w\e[0m\e[0;32m]\n\$> \e[0m"
```
to the `.bashrc` file
